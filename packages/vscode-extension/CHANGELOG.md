# Change Log

All notable changes to the Azodik UI IntelliSense extension will be documented in this file.

## [0.1.0] - 2024

### Added

- Initial release
- IntelliSense for @azodik/ui components
- Hover documentation for components
- Code snippets for common components:
  - Button
  - Input
  - Card
  - Modal
  - Alert
  - Select
  - Theme Provider
- TypeScript and JavaScript support
- Configuration options
