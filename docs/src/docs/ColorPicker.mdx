# ColorPicker

The `ColorPicker` component provides a color selection interface.

## Features
- **Color selection**: Visual color picker interface.
- **Hex values**: Support for hex color codes.
- **Controlled/Uncontrolled**: Both controlled and uncontrolled modes.

## Usage

<CodeBlock language="tsx">
{`import { ColorPicker } from "@azodik/ui";

function App() {
  const [color, setColor] = React.useState("#000000");
  return (
    <ColorPicker
      value={color}
      onChange={setColor}
    />
  );
}`}
</CodeBlock>
