# Center

The `Center` component is a utility component for centering content both horizontally and vertically.

## Features
- **Flexible**: Control horizontal and vertical centering independently.
- **Full dimensions**: Support for full width and full height.

## Usage

<CodeBlock language="tsx">
{`import { Center } from "@azodik/ui";

<Center>
  <div>Centered content</div>
</Center>`}
</CodeBlock>
