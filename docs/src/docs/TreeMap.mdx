# TreeMap

The `TreeMap` component displays hierarchical data as nested rectangles.

## Features
- **Hierarchical visualization**: Show nested data structures.
- **Size encoding**: Rectangle size represents value.
- **Color coding**: Optional color encoding.
- **Interactive**: Click to navigate hierarchy.

## Usage

<CodeBlock language="tsx">
{`import { TreeMap } from "@azodik/ui";

<TreeMap
  data={[
    { id: "1", label: "Category 1", value: 100, children: [
      { id: "1-1", label: "Item 1", value: 50 },
    ]},
  ]}
  width={400}
  height={300}
/>`}
</CodeBlock>
