# Navbar

The `Navbar` component provides a responsive navigation bar with mobile menu support, nested menus, and glass morphism effects.

## Basic Usage

<CodeBlock language="tsx">
{`import { Navbar, NavbarBrand, NavbarContent, NavbarLinks, NavbarLink, NavbarActions, NavbarToggle } from '@azodik/ui';

function App() {
  return (
    <Navbar variant="sticky" isGlass={true}>
      <NavbarBrand href="/">
        My App
      </NavbarBrand>
      
      <NavbarToggle />
      
      <NavbarContent>
        <NavbarLinks>
          <NavbarLink href="/home">Home</NavbarLink>
          <NavbarLink href="/about">About</NavbarLink>
          <NavbarLink href="/contact">Contact</NavbarLink>
        </NavbarLinks>
      </NavbarContent>
      
      <Navbar.Actions>
        <Button>Sign In</Button>
      </Navbar.Actions>
    </Navbar>
  );
}`}
</CodeBlock>

## Nested Menus

{`import { Navbar, NavbarSubmenu } from '@azodik/ui';

  <NavbarContent>
    <NavbarLinks>
      <NavbarLink href="/products">Products</NavbarLink>

</CodeBlock>

## Mobile Menu

The Navbar automatically provides a mobile menu on small screens:

- **Toggle Button**: Appears automatically on mobile
- **Backdrop**: Dark overlay when menu is open
- **Body Scroll Lock**: Prevents scrolling when menu is open
- **Auto-close**: Closes on link click, Escape key, or window resize

## Accessibility

- Full keyboard navigation support
- ARIA labels and roles
- Focus management
- Screen reader friendly

## SSR Compatibility

The Navbar is fully SSR-compatible and works with Next.js, Remix, and other SSR frameworks.
