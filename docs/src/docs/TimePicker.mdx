# TimePicker

The `TimePicker` component provides time selection with hour and minute inputs.

## Features
- **Time selection**: Select hours and minutes.
- **12/24 hour**: Support for 12-hour and 24-hour formats.
- **Seconds**: Optional seconds selection.
- **Validation**: Built-in time validation.

## Usage

<CodeBlock language="tsx">
{`import { TimePicker } from "@azodik/ui";

function App() {
  const [time, setTime] = React.useState("");
  return (
    <TimePicker
      value={time}
      onChange={setTime}
      format12h
    />
  );
}`}
</CodeBlock>
