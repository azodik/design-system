## Font Sizes

We use a numeric scale for font sizes to align with the rest of the design system.

| Token | Size (rem) | Size (px) | Usage |
|-------|------------|-----------|-------|
| `--font-size-xs` | 0.75rem | 12px | Small tags, captions |
| `--font-size-sm` | 0.875rem | 14px | Body text, inputs, buttons |
| `--font-size-md` | 1rem | 16px | Headings, large text |
| `--font-size-lg` | 1.125rem | 18px | Titles |
| `--font-size-xl` | 1.25rem | 20px | Hero text |
