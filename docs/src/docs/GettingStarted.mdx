# Getting Started

Welcome to the Azodik Design System! This comprehensive guide will help you integrate our components into your projects quickly and efficiently.

## Introduction

The Azodik Design System is a modern, accessible, and highly customizable component library built with React and TypeScript. It provides everything you need to build beautiful, consistent user interfaces.

### What's Included

- **üé® Modern Design** - Beautiful orange color palette with dark/light mode
- **‚ö° TypeScript First** - Full TypeScript support with comprehensive type definitions
- **‚ôø Accessible** - Built with accessibility in mind using modern patterns
- **üì± Responsive** - Mobile-first responsive design
- **üéØ Tree Shakeable** - Import only what you need
- **üîß Customizable** - Easy to customize with CSS custom properties
- **üì¶ Lightweight** - Optimized bundle size

## Installation

Install both the Azodik UI React components and the CSS design system. The React components provide the interactive elements, while the CSS package provides the styling foundation.

<CodeBlock language="bash">
{`# Install the React components
npm install @azodik/ui

# Install the CSS design system (required)
npm install azodik-ui-core`}
</CodeBlock>

### Requirements

- React 16.8+ (hooks support)
- TypeScript 4.5+ (optional but recommended)
- Node.js 14+ for development

### Bundle Size

- Tree-shakeable components
- ~19KB CSS + ~35KB React (gzipped)
- Individual component imports

## Quick Start

Once installed, you can start using components in your React application:

<CodeBlock language="jsx">
{`import React from 'react';
import 'azodik-ui-core'; // Import the CSS
import { Button, Alert, Card } from '@azodik/ui';

function App() {
  return (
    <div>
      <Button variant="primary">Click me</Button>
      <Alert variant="success" title="Success!">
        Your changes have been saved.
      </Alert>
      <Card>
        <h3>Welcome to Azodik</h3>
        <p>Start building amazing interfaces!</p>
      </Card>
    </div>
  );
}

export default App;`}
</CodeBlock>

## Theming

### CSS Custom Properties

The design system uses CSS custom properties for easy theming:

<CodeBlock language="css">
{`:root {
  --color-primary: #f97316;
  --color-background: #ffffff;
  --color-text: #111827;
  --color-surface: #f9fafb;
  --color-border: #e5e7eb;
  /* ... more tokens */
}

[data-theme="dark"] {
  --color-background: #0f172a;
  --color-text: #f1f5f9;
  --color-surface: #1e293b;
  --color-border: #334155;
  /* ... dark mode tokens */
}`}
</CodeBlock>

### Theme Switching

<CodeBlock language="jsx">
{`// Toggle theme
function toggleTheme() {
  const html = document.documentElement;
  const currentTheme = html.getAttribute("data-theme");
  const newTheme = currentTheme === "light" ? "dark" : "light";
  
  html.setAttribute("data-theme", newTheme);
  localStorage.setItem("theme", newTheme);
}`}
</CodeBlock>

Import only what you need to keep your bundle size small:

<CodeBlock language="tsx">
{`// ‚úÖ Good - tree shakeable

// ‚ùå Avoid - imports everything
