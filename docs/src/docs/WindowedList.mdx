# WindowedList

The `WindowedList` component renders only visible items in a list for performance with large datasets.

## Features
- **Virtual rendering**: Only render visible items.
- **Large lists**: Handle thousands of items efficiently.
- **Dynamic heights**: Support for fixed or dynamic item heights.
- **Smooth scrolling**: Optimized scrolling performance.

## Usage

<CodeBlock language="tsx">
{`import { WindowedList } from "@azodik/ui";

<WindowedList
  items={items}
  renderItem={(item) => <div>{item.name}</div>}
  itemHeight={50}
  height={400}
/>`}
</CodeBlock>

{`import { WindowedList, Stack } from "@azodik/ui";

export const WindowedListExample = () => {
  const items = Array.from({ length: 1000 }, (_, i) => ({
    id: i,
    name: \`Item \${i + 1}\`
  }));

  return (
    <div style={{ height: "400px" }}>
      <WindowedList
        items={items}
        renderItem={(item) => (
          <div style={{ padding: "var(--space-4)", borderBottom: "1px solid var(--color-border)" }}>
            {item.name}
          </div>
        )}
        itemHeight={50}
        height={400}
