# Flex

The `Flex` component is a layout primitive that helps you build flexible layouts using CSS Flexbox.

## Features
- **Polymorphic**: Can render as any HTML element using the `as` prop.
- **Responsive**: Supports responsive gap and alignment props.

## Usage

<CodeBlock language="tsx">
{`import { Flex } from "@azodik/ui";

<Flex gap="4" align="center">
  <div>Item 1</div>
  <div>Item 2</div>
</Flex>`}
</CodeBlock>

## Responsive Props

All layout props support responsive values. You can pass a single value or an object with breakpoint-specific values:

```tsx
<Flex
  direction={{ base: "column", md: "row" }}
  gap={{ base: "2", md: "4" }}
  align={{ base: "start", md: "center" }}
>
  <div>Item 1</div>
  <div>Item 2</div>
</Flex>
```

Available breakpoints: `base`, `sm` (640px), `md` (768px), `lg` (1024px), `xl` (1280px)
