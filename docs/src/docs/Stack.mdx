# Stack

The `Stack` component is a layout primitive that helps you build flexible layouts using CSS Flexbox with a simpler API than `Flex`.

## Features
- **Responsive**: Supports responsive gap, direction, and alignment props.
- **Simple API**: Easier to use than Flex for common layout patterns.

## Usage

<CodeBlock language="tsx">
{`import { Stack } from "@azodik/ui";

<Stack direction="row" gap={16} align="center">
  <div>Item 1</div>
  <div>Item 2</div>
</Stack>`}
</CodeBlock>

## Responsive Props

All layout props support responsive values. You can pass a single value or an object with breakpoint-specific values:

```tsx
<Stack
  direction={{ base: "column", md: "row" }}
  gap={{ base: 8, md: 16 }}
  align={{ base: "start", md: "center" }}
>
  <div>Item 1</div>
  <div>Item 2</div>
</Stack>
```

Available breakpoints: `base`, `sm` (640px), `md` (768px), `lg` (1024px), `xl` (1280px)
