# Heatmap

The `Heatmap` component displays a calendar heatmap visualization.

## Features
- **Calendar heatmap**: Visualize data over time.
- **Color coding**: Color intensity based on values.
- **Tooltips**: Show data on hover.
- **Customizable**: Cell size, colors, and thresholds.

## Usage

<CodeBlock language="tsx">
{`import { Heatmap } from "@azodik/ui";

<Heatmap
  data={[
    { date: new Date(), value: 10 },
    { date: new Date(), value: 20 },
  ]}
/>`}
</CodeBlock>

### Example

{`import { Heatmap, Stack } from "@azodik/ui";

export const HeatmapExample = () => {
  const data = Array.from({ length: 365 }, (_, i) => ({
    date: new Date(2024, 0, i + 1),
    value: Math.floor(Math.random() * 10),
  }));

  return (
    <Stack gap={16} style={{ width: "100%", padding: "var(--space-4)" }}>
      <Heatmap
        data={data}
        cellSize={12}
        gap={2}
      />
    </Stack>
  );
};`}
</CodeBlockWithPreview>
