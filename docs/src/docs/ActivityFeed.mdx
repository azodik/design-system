# ActivityFeed

The `ActivityFeed` component displays a chronological feed of activities.

## Features
- **Activity timeline**: Show activities in chronological order.
- **Activity types**: Support for different activity types.
- **User avatars**: Show actor information.
- **Metadata**: Support for custom metadata.

## Usage

<CodeBlock language="tsx">
{`import { ActivityFeed } from "@azodik/ui";

<ActivityFeed
  activities={[
    {
      id: "1",
      title: "User logged in",
      timestamp: new Date(),
      type: "info",
    },
  ]}
/>`}
</CodeBlock>
