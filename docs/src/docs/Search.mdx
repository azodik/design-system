# Search

The `Search` component provides a powerful search interface with keyboard shortcuts and recent searches.

## Features
- **Keyboard shortcut**: Open search with Cmd/Ctrl+K.
- **Recent searches**: Remembers recent search history.
- **Customizable**: Custom result rendering and messages.
- **Accessible**: Full keyboard navigation.

## Usage

<CodeBlock language="tsx">
{`import { Search, SearchIndex } from "@azodik/ui";

const searchIndex = new SearchIndex([
  { id: "1", title: "Getting Started", content: "Learn how to get started" },
  { id: "2", title: "Components", content: "Browse all components" },
]);

function App() {
  return (
    <Search
      searchIndex={searchIndex}
      onSelect={(item) => console.log("Selected:", item)}
    />
  );
}`}
</CodeBlock>
