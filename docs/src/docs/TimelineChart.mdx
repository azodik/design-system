# TimelineChart

The `TimelineChart` component displays a timeline chart visualization.

## Features
- **Timeline visualization**: Show data over time.
- **Smooth curves**: Optional smooth curve rendering.
- **Area fill**: Optional area fill under the line.
- **Grid**: Optional grid display.

## Usage

<CodeBlock language="tsx">
{`import { TimelineChart } from "@azodik/ui";

<TimelineChart
  data={[
    { x: new Date(), y: 10 },
    { x: new Date(), y: 20 },
  ]}
  width={400}
  height={200}
/>`}
</CodeBlock>
