# Footer

The `Footer` component provides flexible footer layouts for websites with multiple variants and sub-components.

## Basic Usage

<CodeBlock language="tsx">
{`import { Footer, FooterBrand, FooterContent, FooterLinks, FooterLink, FooterBottom } from '@azodik/ui';

function App() {
  return (
    <Footer variant="simple">
      <FooterBottom>
        © 2024 My App. All rights reserved.
      </FooterBottom>
    </Footer>
  );
}`}
</CodeBlock>

## Variants

<CodeBlock language="tsx">

<Footer variant="centered">...</Footer>`}
</CodeBlock>

## Sub-components

- **FooterBrand**: Brand/logo section
- **FooterContent**: Main content container (for columns layout)
- **FooterLinks**: Group of links with optional title
- **FooterLink**: Individual footer link
- **FooterBottom**: Bottom section (copyright, etc.)
- **FooterSocial**: Social media links container

## Responsive

The Footer component is fully responsive and adapts to different screen sizes. On mobile, multi-column layouts stack vertically.
