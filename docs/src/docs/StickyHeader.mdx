# StickyHeader

The `StickyHeader` component creates a header that sticks to the top (or bottom) when scrolling.

## Features
- **Sticky positioning**: Header sticks when scrolling.
- **Auto-hide**: Optionally hide on scroll down, show on scroll up.
- **Threshold**: Configurable scroll threshold.
- **Customizable**: Position and z-index control.

## Usage

<CodeBlock language="tsx">
{`import { StickyHeader } from "@azodik/ui";

<StickyHeader autoHide threshold={100}>
  <Navigation />
</StickyHeader>`}
</CodeBlock>
