# TagInput

The `TagInput` component allows users to input multiple tags.

## Features
- **Multiple tags**: Add and remove multiple tags.
- **Validation**: Custom tag validation.
- **Removable**: Tags can be removed individually.
- **Keyboard support**: Enter and comma to add tags.

## Usage

<CodeBlock language="tsx">
{`import { TagInput } from "@azodik/ui";

function App() {
  const [tags, setTags] = React.useState([]);
  return (
    <TagInput
      value={tags}
      onChange={setTags}
      placeholder="Add tags"
    />
  );
}`}
</CodeBlock>
