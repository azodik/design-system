# Section

The `Section` component provides consistent vertical padding for content sections. It works well with the `Container` component.

## Features
- **Consistent spacing**: Provides standardized vertical padding.
- **Polymorphic**: Can render as any HTML element using the `as` prop.
- **Sized**: Supports size variants for different padding amounts.

## Usage

<CodeBlock language="tsx">
{`import { Section, Container } from "@azodik/ui";

<Section>
  <Container size="md">
    <div>Section content</div>
  </Container>
</Section>`}
</CodeBlock>
