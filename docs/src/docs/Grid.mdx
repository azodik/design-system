# Grid

The `Grid` component is a layout primitive that helps you build grid layouts using CSS Grid.

## Usage

<CodeBlock language="tsx">
{`import { Grid } from "@azodik/ui";

<Grid columns="3" gap="4">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</Grid>`}
</CodeBlock>
