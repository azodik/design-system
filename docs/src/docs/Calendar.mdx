# Calendar

The `Calendar` component provides a date picker interface for selecting dates.

## Features
- **Date selection**: Single date selection with validation.
- **Date constraints**: Min/max date and disabled dates support.
- **Customizable**: Size variants and week number display.
- **Accessible**: Full keyboard navigation and ARIA support.

## Usage

<CodeBlock language="tsx">
{`import { Calendar } from "@azodik/ui";

function App() {
  const [date, setDate] = React.useState(new Date());
  return (
    <Calendar
      value={date}
      onChange={setDate}
    />
  );
}`}
</CodeBlock>
