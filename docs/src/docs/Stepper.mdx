# Stepper

The `Stepper` component displays a multi-step process indicator.

## Features
- **Step tracking**: Visual progress through steps.
- **Orientation**: Horizontal or vertical layout.
- **Interactive**: Click to navigate steps.
- **Customizable**: Show numbers, labels, and descriptions.

## Usage

<CodeBlock language="tsx">
{`import { Stepper } from "@azodik/ui";

<Stepper
  steps={[
    { label: "Step 1", completed: true },
    { label: "Step 2", active: true },
    { label: "Step 3" },
  ]}
/>`}
</CodeBlock>
