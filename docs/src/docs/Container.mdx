# Container

The Container component constrains the maximum width of content. It ensures your content remains readable on large screens.

<Flex gap="2" style={{ marginBottom: "var(--space-4)" }}>
  <Badge variant="solid" color="indigo">Layout</Badge>
  <Badge variant="outline" color="neutral">Responsive</Badge>
</Flex>

## Usage

<CodeBlock language="tsx">
{`import { Container } from "@azodik/ui";

function App() {
  return (
    <Container size="md">
      <div>My content</div>
    </Container>
  );
}`}
</CodeBlock>

### Section

The layout package also exports a `Section` component that goes well with `Container`. `Section` provides consistent vertical padding.

<CodeBlockWithPreview
  preview={
    <div style={{ width: "100%", border: "1px solid var(--color-border)", borderRadius: "var(--radius-2)", overflow: "hidden" }}>
      <div style={{ backgroundColor: "var(--gray-2)" }}>
        <Container size="md">
          <div style={{ paddingTop: "var(--space-8)", paddingBottom: "var(--space-8)", textAlign: "center" }}>Hero Section</div>
        </Container>
      </div>
      <Container size="md">
        <Center style={{ paddingTop: "var(--space-12)", paddingBottom: "var(--space-12)", borderLeft: "1px dashed var(--gray-6)", borderRight: "1px dashed var(--gray-6)", height: "8rem" }}>
          Content
        </Center>
      </Container>
    </div>
  }
  code={`import { Container, Center } from "@azodik/ui";

export const ContainerSection = () => {
  return (
    <div style={{ width: "100%", border: "1px solid var(--color-border)", borderRadius: "var(--radius-2)", overflow: "hidden" }}>
      <div style={{ backgroundColor: "var(--gray-2)" }}>
        <Container size="md">
          <div style={{ paddingTop: "var(--space-8)", paddingBottom: "var(--space-8)", textAlign: "center" }}>Hero Section</div>
        </Container>
      </div>
      <Container size="md">
        <Center style={{ paddingTop: "var(--space-12)", paddingBottom: "var(--space-12)", borderLeft: "1px dashed var(--gray-6)", borderRight: "1px dashed var(--gray-6)", height: "8rem" }}>
          Content
        </Center>
      </Container>
    </div>
  );
};`}
/>
