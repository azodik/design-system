# Tour

The `Tour` component provides an interactive guided tour for onboarding users.

## Features
- **Step-by-step**: Guide users through features.
- **Target elements**: Highlight specific elements.
- **Navigation**: Previous/next step navigation.
- **Customizable**: Custom actions and content.

## Usage

<CodeBlock language="tsx">
{`import { Tour, Button } from "@azodik/ui";

function App() {
  const [isActive, setIsActive] = React.useState(false);
  return (
    <>
      <Button onClick={() => setIsActive(true)}>Start Tour</Button>
      <Tour
        isActive={isActive}
        steps={[
          { id: "1", title: "Welcome", content: "This is step 1" },
          { id: "2", title: "Feature", content: "This is step 2" },
        ]}
        onEnd={() => setIsActive(false)}
      />
    </>
  );
}`}
</CodeBlock>
