# BottomSheet

The `BottomSheet` component displays a mobile-friendly slide-up sheet from the bottom.

## Features
- **Mobile optimized**: Designed for mobile interactions.
- **Smooth animation**: Slide-up animation.
- **Scrollable**: Content can scroll if needed.
- **Accessible**: Full keyboard and screen reader support.

## Usage

<CodeBlock language="tsx">
{`import { BottomSheet, Button } from "@azodik/ui";

function App() {
  const [open, setOpen] = React.useState(false);
  return (
    <>
      <Button onClick={() => setOpen(true)}>Open Bottom Sheet</Button>
      <BottomSheet open={open} onOpenChange={setOpen} title="Settings">
        <div>Content here</div>
      </BottomSheet>
    </>
  );
}`}
</CodeBlock>
