# Timeline

The `Timeline` component displays chronological events in a timeline format.

## Features
- **Chronological display**: Show events in order.
- **Icons and colors**: Custom icons and color variants.
- **Orientation**: Vertical or horizontal layout.
- **Status indicators**: Active and completed states.

## Usage

<CodeBlock language="tsx">
{`import { Timeline } from "@azodik/ui";

<Timeline
  items={[
    { title: "Step 1", description: "Completed", completed: true },
    { title: "Step 2", description: "Active", active: true },
    { title: "Step 3", description: "Pending" },
  ]}
/>`}
</CodeBlock>
