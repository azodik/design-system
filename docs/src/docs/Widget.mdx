# Widget

The `Widget` component provides a reusable dashboard widget wrapper.

## Features
- **Dashboard widget**: Standardized widget container.
- **Header and footer**: Optional header and footer sections.
- **Actions**: Support for action buttons.
- **Loading and error states**: Built-in state handling.

## Usage

<CodeBlock language="tsx">
{`import { Widget } from "@azodik/ui";

<Widget
  title="Sales Overview"
  description="Monthly sales data"
  actions={[
    { label: "Export", onClick: handleExport },
  ]}
>
  <div>Widget content</div>
</Widget>`}
</CodeBlock>
