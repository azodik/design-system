# PhoneInput

The `PhoneInput` component provides phone number input with country code selection.

## Features
- **Country selector**: Select country code.
- **Formatting**: Automatic phone number formatting.
- **Validation**: Built-in phone number validation.
- **International**: Support for international formats.

## Usage

<CodeBlock language="tsx">
{`import { PhoneInput } from "@azodik/ui";

function App() {
  const [phone, setPhone] = React.useState("");
  return (
    <PhoneInput
      value={phone}
      onChange={(phone, countryCode) => {
        setPhone(phone);
        console.log("Country:", countryCode);
      }}
      defaultCountry="US"
    />
  );
}`}
</CodeBlock>
