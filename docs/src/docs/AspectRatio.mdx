# AspectRatio

The `AspectRatio` component maintains a consistent aspect ratio for its children, useful for images and videos.

## Features
- **Ratio control**: Maintains aspect ratio using padding technique.
- **Presets**: Common aspect ratio presets (square, video, photo, etc.).
- **Custom ratios**: Support for custom numeric ratios.

## Usage

<CodeBlock language="tsx">
{`import { AspectRatio } from "@azodik/ui";

<AspectRatio ratio={16/9}>
  <img src="image.jpg" alt="Image" />
</AspectRatio>`}
</CodeBlock>
