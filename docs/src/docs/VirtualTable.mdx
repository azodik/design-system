# VirtualTable

The `VirtualTable` component provides virtual scrolling for large datasets, rendering only visible rows.

## Features
- **Virtual scrolling**: Only render visible rows for performance.
- **Large datasets**: Handle thousands of rows efficiently.
- **Sorting**: Support for column sorting.
- **Customizable**: All Table component features.

## Usage

<CodeBlock language="tsx">
{`import { VirtualTable } from "@azodik/ui";

<VirtualTable
  data={largeDataset}
  columns={columns}
  height={600}
  estimatedRowHeight={50}
/>`}
</CodeBlock>

{`import { VirtualTable, Stack } from "@azodik/ui";

export const VirtualTableExample = () => {
  const data = Array.from({ length: 1000 }, (_, i) => ({
    id: i + 1,
    name: \`Item \${i + 1}\`,
    value: Math.random() * 100
  }));

  const columns = [
    { key: "id", label: "ID", sortable: true },
    { key: "name", label: "Name", sortable: true },
    { key: "value", label: "Value", sortable: true },
  ];

  return (
    <div style={{ height: "400px" }}>
      <VirtualTable
        data={data}
        columns={columns}
        height={400}
        estimatedRowHeight={40}
      />
    </div>
  );
};`}
