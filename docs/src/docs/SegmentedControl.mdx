# SegmentedControl

The `SegmentedControl` component provides iOS-style segmented control for selecting between multiple options.

## Features
- **Multiple options**: Select from multiple options.
- **Icons**: Support for icons in options.
- **Full width**: Optional full-width layout.
- **Accessible**: Full keyboard navigation.

## Usage

<CodeBlock language="tsx">
{`import { SegmentedControl } from "@azodik/ui";

function App() {
  const [value, setValue] = React.useState("option1");
  return (
    <SegmentedControl
      options={[
        { value: "option1", label: "Option 1" },
        { value: "option2", label: "Option 2" },
      ]}
      value={value}
      onChange={setValue}
    />
  );
}`}
</CodeBlock>
