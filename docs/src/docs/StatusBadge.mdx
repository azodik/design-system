# StatusBadge

The `StatusBadge` component displays status indicators with color coding.

## Features
- **Status types**: Multiple status types (online, offline, success, error, etc.).
- **Pulse animation**: Optional pulse animation.
- **Labels**: Show or hide status labels.
- **Customizable**: Size variants.

## Usage

<CodeBlock language="tsx">
{`import { StatusBadge } from "@azodik/ui";

<StatusBadge status="online" label="Online" />
<StatusBadge status="offline" />
<StatusBadge status="success" label="Success" />`}
</CodeBlock>
