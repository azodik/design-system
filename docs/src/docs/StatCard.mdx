# StatCard

The `StatCard` component displays key performance indicators (KPIs) with trend indicators.

## Features
- **KPI display**: Show metrics with values.
- **Trend indicators**: Show percentage changes and trends.
- **Icons**: Support for custom icons.
- **Loading state**: Skeleton loading support.

## Usage

<CodeBlock language="tsx">
{`import { StatCard } from "@azodik/ui";

<StatCard
  title="Total Revenue"
  value="$12,345"
  change={12.5}
  trend="up"
/>`}
</CodeBlock>
