# FileUpload

The `FileUpload` component provides file upload functionality with drag and drop support.

## Features
- **Drag and drop**: Support for drag and drop file uploads.
- **File validation**: Size and type validation.
- **Multiple files**: Support for multiple file selection.
- **Preview**: File preview support.

## Usage

<CodeBlock language="tsx">
{`import { FileUpload } from "@azodik/ui";

function App() {
  return (
    <FileUpload
      accept="image/*"
      onFilesSelected={(files) => console.log(files)}
      dragAndDrop
    />
  );
}`}
</CodeBlock>
