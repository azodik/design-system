# MetricCard

The `MetricCard` component displays simple key performance indicators (KPIs).

## Features
- **Simple KPI**: Display label and value.
- **Icons**: Support for custom icons.
- **Loading state**: Skeleton loading support.
- **Clickable**: Optional click handler.

## Usage

<CodeBlock language="tsx">
{`import { MetricCard } from "@azodik/ui";

<MetricCard
  label="Total Users"
  value="1,234"
  icon={<UsersIcon />}
/>`}
</CodeBlock>
