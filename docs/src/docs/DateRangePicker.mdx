# DateRangePicker

The `DateRangePicker` component allows users to select a date range.

## Features
- **Date range**: Select start and end dates.
- **Quick select**: Predefined date range options.
- **Validation**: Date range validation.

## Usage

<CodeBlock language="tsx">
{`import { DateRangePicker } from "@azodik/ui";

function App() {
  const [range, setRange] = React.useState({ start: null, end: null });
  return (
    <DateRangePicker
      value={range}
      onChange={setRange}
      showQuickSelect
    />
  );
}`}
</CodeBlock>
