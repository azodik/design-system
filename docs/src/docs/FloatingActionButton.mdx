# FloatingActionButton

The `FloatingActionButton` (FAB) is a circular button that floats above the content for primary actions.

## Features
- **Floating**: Positioned absolutely on the screen.
- **Extended**: Support for extended FAB with label.
- **Positions**: Multiple position options.
- **Accessible**: Full keyboard and screen reader support.

## Usage

<CodeBlock language="tsx">
{`import { FloatingActionButton } from "@azodik/ui";

<FloatingActionButton
  icon={<PlusIcon />}
  position="bottom-right"
  onClick={() => console.log("Clicked")}
/>`}
</CodeBlock>
