# Slider

The `Slider` component allows users to select a numeric value from a range by dragging a handle.

## Features
- **Range support**: Single value or dual-handle range selection.
- **Customizable**: Size, color, and value formatting options.
- **Accessible**: Full keyboard navigation and ARIA support.

## Usage

<CodeBlock language="tsx">
{`import { Slider } from "@azodik/ui";

function App() {
  const [value, setValue] = React.useState(50);
  return (
    <Slider
      min={0}
      max={100}
      value={value}
      onChange={setValue}
    />
  );
}`}
</CodeBlock>
