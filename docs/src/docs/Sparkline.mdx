# Sparkline

The `Sparkline` component displays mini inline charts for data visualization.

## Features
- **Mini charts**: Small inline charts.
- **Smooth curves**: Optional smooth curve rendering.
- **Dots**: Show data point markers.
- **Fill area**: Optional area fill under the line.

## Usage

<CodeBlock language="tsx">
{`import { Sparkline } from "@azodik/ui";

<Sparkline
  data={[10, 20, 15, 30, 25, 40]}
  width={100}
  height={30}
  color="#f97316"
/>`}
</CodeBlock>
