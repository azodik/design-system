# Menu

The `Menu` component displays a dropdown menu with a list of actions.

## Features
- **Dropdown menu**: Context menu with items and actions.
- **Submenus**: Support for nested menu items.
- **Icons and shortcuts**: Rich menu item content.
- **Accessible**: Full keyboard navigation.

## Usage

<CodeBlock language="tsx">
{`import { Menu, Button } from "@azodik/ui";

function App() {
  return (
    <Menu
      trigger={<Button>Open Menu</Button>}
      items={[
        { label: "Edit", onClick: () => console.log("Edit") },
        { label: "Delete", onClick: () => console.log("Delete"), destructive: true },
      ]}
    />
  );
}`}
</CodeBlock>
