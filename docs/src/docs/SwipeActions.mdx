# SwipeActions

The `SwipeActions` component enables swipe gestures on mobile to reveal action buttons.

## Features
- **Swipe gestures**: Swipe left or right to reveal actions.
- **Multiple actions**: Support for multiple action buttons.
- **Color coding**: Color-coded actions.
- **Mobile optimized**: Designed for touch interactions.

## Usage

<CodeBlock language="tsx">
{`import { SwipeActions } from "@azodik/ui";

<SwipeActions
  rightActions={[
    { label: "Delete", color: "ruby", onClick: handleDelete },
    { label: "Archive", color: "amber", onClick: handleArchive }
  ]}
>
  <div>List item content</div>
</SwipeActions>`}
</CodeBlock>
