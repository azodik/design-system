# BottomNavigation

The `BottomNavigation` component provides a mobile-friendly tab bar at the bottom of the screen.

## Features
- **Mobile navigation**: Bottom tab bar for mobile apps.
- **Icons and labels**: Support for icons and labels.
- **Badges**: Show badge counts on items.
- **Fixed position**: Optional fixed positioning.

## Usage

<CodeBlock language="tsx">
{`import { BottomNavigation } from "@azodik/ui";

<BottomNavigation
  items={[
    { value: "home", label: "Home", icon: <HomeIcon /> },
    { value: "search", label: "Search", icon: <SearchIcon /> },
  ]}
  value="home"
  onChange={(value) => console.log(value)}
/>`}
</CodeBlock>
