# ContextMenu

The `ContextMenu` component displays a context menu on right-click.

## Features
- **Right-click menu**: Context menu triggered by right-click.
- **Menu items**: Support for icons, separators, and submenus.
- **Accessible**: Full keyboard navigation.

## Usage

<CodeBlock language="tsx">
{`import { ContextMenu } from "@azodik/ui";

<ContextMenu
  items={[
    { label: "Edit", onClick: () => console.log("Edit") },
    { label: "Delete", onClick: () => console.log("Delete"), destructive: true },
  ]}
>
  <div>Right-click me</div>
</ContextMenu>`}
</CodeBlock>
