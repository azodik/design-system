# Sheet

The `Sheet` component displays a slide-up sheet panel, similar to a drawer but optimized for mobile.

## Features
- **Slide animation**: Smooth slide-up/down animation.
- **Multiple positions**: Top, bottom, left, right.
- **Sizes**: Multiple size variants.
- **Accessible**: Full keyboard and screen reader support.

## Usage

<CodeBlock language="tsx">
{`import { Sheet, Button } from "@azodik/ui";

function App() {
  const [open, setOpen] = React.useState(false);
  return (
    <>
      <Button onClick={() => setOpen(true)}>Open Sheet</Button>
      <Sheet open={open} onOpenChange={setOpen} title="Settings">
        <div>Sheet content</div>
      </Sheet>
    </>
  );
}`}
</CodeBlock>
