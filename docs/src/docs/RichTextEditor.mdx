# RichTextEditor

The `RichTextEditor` component provides a WYSIWYG rich text editing interface.

## Features
- **WYSIWYG editing**: What you see is what you get.
- **Toolbar**: Formatting toolbar with common options.
- **HTML output**: Outputs HTML content.
- **Customizable**: Custom toolbar buttons.

## Usage

<CodeBlock language="tsx">
{`import { RichTextEditor } from "@azodik/ui";

function App() {
  const [content, setContent] = React.useState("");
  return (
    <RichTextEditor
      value={content}
      onChange={setContent}
      placeholder="Start typing..."
    />
  );
}`}
</CodeBlock>
