# Charts Component

The Charts component provides a comprehensive set of data visualization components built on top of Recharts. It includes Line, Area, Bar, and Pie charts with customizable styling and interactive features.

## Introduction

The Charts component includes several complementary features to handle various data visualization and analytics scenarios:

- Charts: the main wrapper for creating comprehensive data visualization components built on Recharts.
- Chart Types: multiple chart variants (Line, Area, Bar, Pie) for different data representation needs.
- Chart Customization: flexible colors, sizing, titles, and styling options for brand-specific designs.
- Chart Interactivity: built-in tooltips, legends, and hover effects to enhance data exploration.
- Chart Responsiveness: adaptive sizing and mobile-friendly layouts for different screen sizes.
- Chart Accessibility: proper ARIA attributes and keyboard navigation for screen reader support.

### When to use
- Display trends over time (Line charts)
- Show cumulative data (Area charts)
- Compare values across categories (Bar charts)
- Show proportional data (Pie charts)
- Create dashboards and analytics interfaces
- Visualize business metrics and KPIs

## Usage

<CodeBlock language="tsx">
{`import { LineChart, AreaChart, BarChart, PieChart } from '@azodik/ui';`}
</CodeBlock>

## Data Format

All charts expect data in the following format:

<CodeBlock language="typescript">
{`interface ChartData {
  name: string;        // X-axis label or category name
  value?: number;      // Y-axis value (for most charts)
  [key: string]: any;  // Additional data properties
}`}
</CodeBlock>

## Accessibility

- Charts are built with accessibility in mind using Recharts
- Screen readers can access chart data through proper ARIA attributes
- Keyboard navigation is supported for interactive elements
- Color is not the only way to convey information
- Tooltips provide additional context for data points

## Design Guidelines

- Use appropriate chart types for your data
- Keep titles and labels concise and descriptive
- Use consistent colors across related charts
- Ensure sufficient contrast for readability
- Consider responsive design for different screen sizes
- Use legends when multiple data series are present
