# SplitPane

The `SplitPane` component creates resizable split panels for side-by-side layouts.

## Features
- **Resizable**: Drag to resize panels.
- **Direction**: Horizontal or vertical split.
- **Constraints**: Min/max size limits.
- **Customizable**: Handle size and visibility.

## Usage

<CodeBlock language="tsx">
{`import { SplitPane } from "@azodik/ui";

<SplitPane
  left={<div>Left Panel</div>}
  right={<div>Right Panel</div>}
  defaultSplit={50}
/>`}
</CodeBlock>
