# Rating

The `Rating` component displays a star rating system for user feedback and reviews.

## Features
- **Interactive**: Click to set rating value.
- **Half stars**: Support for half-star ratings.
- **Read-only**: Display mode for showing ratings.
- **Customizable**: Size and color variants.

## Usage

<CodeBlock language="tsx">
{`import { Rating } from "@azodik/ui";

function App() {
  const [rating, setRating] = React.useState(0);
  return (
    <Rating
      value={rating}
      onChange={setRating}
      max={5}
    />
  );
}`}
</CodeBlock>
