# Gauge

The `Gauge` component displays a circular gauge/progress indicator.

## Features
- **Circular gauge**: Visual gauge display.
- **Value display**: Show current value.
- **Variants**: Full, half, or quarter circle.
- **Customizable**: Size, color, and thickness.

## Usage

<CodeBlock language="tsx">
{`import { Gauge } from "@azodik/ui";

<Gauge
  value={75}
  min={0}
  max={100}
  size={120}
  showValue
/>`}
</CodeBlock>
