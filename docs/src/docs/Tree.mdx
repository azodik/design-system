# Tree

The `Tree` component displays hierarchical data in a tree structure.

## Features
- **Hierarchical data**: Display nested tree structures.
- **Expand/collapse**: Expand and collapse nodes.
- **Selection**: Single or multiple node selection.
- **Icons**: Support for custom node icons.

## Usage

<CodeBlock language="tsx">
{`import { Tree } from "@azodik/ui";

<Tree
  data={[
    { id: "1", label: "Node 1", children: [
      { id: "1-1", label: "Child 1" },
      { id: "1-2", label: "Child 2" },
    ]},
  ]}
/>`}
</CodeBlock>
