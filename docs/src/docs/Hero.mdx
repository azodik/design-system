# Hero

The `Hero` component provides a flexible hero section for landing pages with multiple variants and sizes.

## Basic Usage

<CodeBlock language="tsx">
{`import { Hero, HeroTitle, HeroDescription, HeroActions, Button } from '@azodik/ui';

function LandingPage() {
  return (
    <Hero variant="centered" size="large">
      <HeroTitle>Welcome to Our Platform</HeroTitle>
      <HeroDescription>
        Build amazing applications with our design system
      </HeroDescription>
      <HeroActions>
        <Button variant="solid">Get Started</Button>
        <Button variant="outline">Learn More</Button>
      </HeroActions>
    </Hero>
  );
}`}
</CodeBlock>

### Split Layout

<CodeBlock language="tsx">
{`<Hero variant="split" size="large">
  <div>
    <HeroTitle>Split Hero Layout</HeroTitle>
    <HeroDescription>
      Content on the left, image or visual on the right
    </HeroDescription>
    <HeroActions>
      <Button variant="solid">Get Started</Button>
    </HeroActions>
  </div>
  <div>
    {/* Image or visual content */}
  </div>
</Hero>`}
</CodeBlock>

### Subtle Variant

## Sizes

<CodeBlock language="tsx">
{`// Small hero

<Hero size="medium">...</Hero>

// Large hero
<Hero size="large">...</Hero>

// Full viewport height
<Hero size="full">...</Hero>`}
</CodeBlock>

## Sub-components

- **HeroTitle**: Main heading (h1)
- **HeroDescription**: Descriptive text (p)
- **HeroActions**: Container for action buttons

## Responsive

The Hero component is fully responsive and adapts to different screen sizes automatically.
