# Checkbox Component

The Checkbox component provides binary choice controls for enabling or disabling options. It's perfect for forms that require multiple selections or toggles.

## Overview

Checkboxes allow users to select one or more options from a set of choices. They provide clear visual feedback for selected states and are essential for forms and settings.

### When to use
- Multiple selections from a list
- Enabling or disabling features
- Agreeing to terms and conditions
- Filtering options
- Settings and preferences

## Accessibility

- Checkboxes have proper labels and associations
- Screen readers can identify checkbox states
- Keyboard navigation is supported
- Focus management is handled correctly
- Proper ARIA attributes are applied

## Design Guidelines

- Always provide clear labels
- Use consistent sizing across your application
- Provide clear visual feedback for all states
- Group related checkboxes together
- Ensure sufficient touch targets on mobile
