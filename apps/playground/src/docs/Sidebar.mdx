# Sidebar Component

The Sidebar component provides vertical navigation panels for app layouts and navigation. It's perfect for creating consistent navigation experiences in web applications.

## Overview

Sidebars are vertical navigation panels that provide consistent access to different sections of your application. They help users navigate and understand the structure of your app.

### When to use
- Main application navigation
- Dashboard layouts
- Admin interfaces
- Multi-section applications
- Desktop-focused applications

### When not to use
- For mobile-first applications
- When horizontal space is limited
- For simple single-page applications
- When you need horizontal navigation

## Collapsible Sidebar

<CodeBlockWithPreview 
  title="Collapsible Sidebar"
  preview={
    <div className="flex">
      <Sidebar
        items={[
          { label: 'Dashboard', href: '/dashboard', icon: '游' },
          { label: 'Users', href: '/users', icon: '游논' },
          { label: 'Settings', href: '/settings', icon: '丘뙖잺' },
          { label: 'Reports', href: '/reports', icon: '游늵' }
        ]}
        collapsed={false}
        onToggle={() => console.log('Toggle sidebar')}
      />
      
      <div className="flex-1 p-4">
        <button className="px-4 py-2 bg-blue-500 text-white rounded">
          Toggle Sidebar
        </button>
        <p>Main content area</p>
      </div>
    </div>
  }
>
{`import { Sidebar, Button } from '@azodik/ui';
import { useState } from 'react';

export const CollapsibleSidebarExample = () => {
  const [isCollapsed, setIsCollapsed] = useState(false);
  
  const sidebarItems = [
    { label: 'Dashboard', href: '/dashboard', icon: '游' },
    { label: 'Users', href: '/users', icon: '游논' },
    { label: 'Settings', href: '/settings', icon: '丘뙖잺' },
    { label: 'Reports', href: '/reports', icon: '游늵' }
  ];

  return (
    <div className="flex">
      <Sidebar
        items={sidebarItems}
        collapsed={isCollapsed}
        onToggle={() => setIsCollapsed(!isCollapsed)}
      />
      
      <div className="flex-1 p-4">
        <Button onClick={() => setIsCollapsed(!isCollapsed)}>
          Toggle Sidebar
        </Button>
        <p>Main content area</p>
      </div>
    </div>
  );
};`}
</CodeBlockWithPreview>

## Accessibility

- Sidebar has proper semantic structure
- Screen readers can navigate menu items
- Keyboard navigation is supported
- Focus management is handled correctly
- Proper ARIA attributes are applied

## Design Guidelines

- Use clear and descriptive navigation labels
- Keep the number of top-level items reasonable
- Provide clear visual feedback for active states
- Use consistent icons and styling
- Consider mobile responsiveness
