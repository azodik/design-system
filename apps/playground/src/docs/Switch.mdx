# Switch Component

The Switch component provides toggle switches for binary on/off states and settings. It's perfect for enabling or disabling features with immediate visual feedback.

## Overview

Switches are toggle controls that allow users to enable or disable features with a single action. They provide immediate visual feedback and are ideal for settings and preferences.

### When to use
- Enabling or disabling features
- Settings and preferences
- Binary on/off states
- Quick toggles for options
- Mobile-friendly controls

### When not to use
- For multiple selections (use Checkbox instead)
- For single choice from multiple options (use Radio instead)
- For primary actions (use Button instead)
- When you need to show intermediate states

## With Controlled State

<CodeBlockWithPreview 
  title="Controlled Switch"
  preview={
    <div className="flex items-center space-x-2">
      <Switch
        id="dark-mode"
        checked={false}
        onChange={() => console.log('Switch toggled')}
      />
      <label htmlFor="dark-mode">Dark mode</label>
    </div>
  }
>
{`import { Switch } from '@azodik/ui';
import { useState } from 'react';

export const ControlledSwitchExample = () => {
  const [isEnabled, setIsEnabled] = useState(false);

  const handleChange = (checked) => {
    setIsEnabled(checked);
  };

  return (
    <div className="flex items-center space-x-2">
      <Switch
        id="dark-mode"
        checked={isEnabled}
        onChange={handleChange}
      />
      <label htmlFor="dark-mode">Dark mode</label>
    </div>
  );
};`}
</CodeBlockWithPreview>

## Multiple Switches

<CodeBlockWithPreview 
  title="Multiple Switches"
  preview={
    <div className="space-y-4">
      <h3>Settings</h3>
      
      <div className="flex items-center justify-between">
        <label htmlFor="notifications">Push Notifications</label>
        <Switch
          id="notifications"
          checked={false}
          onChange={() => console.log('Notifications toggled')}
        />
      </div>
      
      <div className="flex items-center justify-between">
        <label htmlFor="dark-mode">Dark Mode</label>
        <Switch
          id="dark-mode"
          checked={true}
          onChange={() => console.log('Dark mode toggled')}
        />
      </div>
      
      <div className="flex items-center justify-between">
        <label htmlFor="auto-save">Auto Save</label>
        <Switch
          id="auto-save"
          checked={false}
          onChange={() => console.log('Auto save toggled')}
        />
      </div>
      
      <div className="flex items-center justify-between">
        <label htmlFor="analytics">Analytics</label>
        <Switch
          id="analytics"
          checked={true}
          onChange={() => console.log('Analytics toggled')}
        />
      </div>
    </div>
  }
>
{`import { Switch } from '@azodik/ui';
import { useState } from 'react';

export const MultipleSwitchesExample = () => {
  const [settings, setSettings] = useState({
    notifications: false,
    darkMode: true,
    autoSave: false,
    analytics: true
  });

  const handleChange = (setting, checked) => {
    setSettings(prev => ({
      ...prev,
      [setting]: checked
    }));
  };

  return (
    <div className="space-y-4">
      <h3>Settings</h3>
      
      <div className="flex items-center justify-between">
        <label htmlFor="notifications">Push Notifications</label>
        <Switch
          id="notifications"
          checked={settings.notifications}
          onChange={(checked) => handleChange('notifications', checked)}
        />
      </div>
      
      <div className="flex items-center justify-between">
        <label htmlFor="dark-mode">Dark Mode</label>
        <Switch
          id="dark-mode"
          checked={settings.darkMode}
          onChange={(checked) => handleChange('darkMode', checked)}
        />
      </div>
      
      <div className="flex items-center justify-between">
        <label htmlFor="auto-save">Auto Save</label>
        <Switch
          id="auto-save"
          checked={settings.autoSave}
          onChange={(checked) => handleChange('autoSave', checked)}
        />
      </div>
      
      <div className="flex items-center justify-between">
        <label htmlFor="analytics">Analytics</label>
        <Switch
          id="analytics"
          checked={settings.analytics}
          onChange={(checked) => handleChange('analytics', checked)}
        />
      </div>
    </div>
  );
};`}
</CodeBlockWithPreview>

## Accessibility

- Switches have proper labels and associations
- Screen readers can identify switch states
- Keyboard navigation is supported
- Focus management is handled correctly
- Proper ARIA attributes are applied

## Design Guidelines

- Always provide clear labels
- Use consistent sizing across your application
- Provide clear visual feedback for all states
- Group related switches together
- Ensure sufficient touch targets on mobile
