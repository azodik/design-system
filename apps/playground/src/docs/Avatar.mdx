# Avatar Component

The Avatar component displays user profile images, initials, or placeholder icons. It provides a consistent way to represent users throughout your application.

## Overview

Avatars are used to represent users in your application. They can display profile images, user initials, or fallback icons when no image is available.

### When to use
- Display user profile pictures
- Show user initials when no image is available
- Represent users in lists, comments, or user interfaces
- Create consistent user representation across the app

## Accessibility

- Avatars have appropriate alt text for screen readers
- Initials are generated from the name prop for better accessibility
- Color contrast meets WCAG guidelines
- Focus states are provided for interactive avatars

## Design Guidelines

- Use consistent sizing across your application
- Provide meaningful alt text for images
- Use initials as fallback when images are not available
- Consider the context when choosing avatar sizes
- Ensure avatars are recognizable and not too small
