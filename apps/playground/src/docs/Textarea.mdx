# Textarea Component

The Textarea component provides multi-line text input for longer content and messages. It's perfect for collecting detailed user input and comments.

## Introduction

The Textarea component includes several complementary features to handle various multi-line text input scenarios:

- Textarea: the main wrapper for creating multi-line text input fields with larger input areas for longer content.
- Textarea Sizing: flexible sizing options with auto-resize capabilities and configurable min/max rows for different content needs.
- Textarea Validation: built-in validation support with error messages, success states, and form integration capabilities.
- Textarea Accessibility: proper ARIA attributes, keyboard navigation, and screen reader support for multi-line text inputs.
- Textarea Customization: flexible styling, placeholder text, and appearance options for different design contexts.
- Textarea States: support for disabled, readonly, and interactive states with appropriate visual feedback and user guidance.

### When to use
- Collecting longer text input
- Comment forms and feedback
- Description fields
- Message composition
- Multi-line content entry
