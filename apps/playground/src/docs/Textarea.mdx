# Textarea Component

The Textarea component provides multi-line text input for longer content and messages. It's perfect for collecting detailed user input and comments.

## Overview

Textarea components are used for collecting multi-line text input from users. They provide a larger input area for longer content like comments, descriptions, or messages.

### When to use
- Collecting longer text input
- Comment forms and feedback
- Description fields
- Message composition
- Multi-line content entry

## Accessibility

- Textareas have proper labels and associations
- Error messages are announced to screen readers
- Focus management is handled correctly
- Keyboard navigation is supported
- Color contrast meets WCAG guidelines

## Design Guidelines

- Always provide clear labels
- Use appropriate placeholder text
- Show character limits when applicable
- Provide clear validation feedback
- Ensure sufficient touch targets on mobile
