# Breadcrumb Component

The Breadcrumb component shows the current page location within the site hierarchy. It helps users understand where they are and navigate back to previous levels.

## Overview

Breadcrumbs provide a navigation trail that shows users their current location within a website or application. They help users understand the site structure and navigate back to previous levels.

### When to use
- Show hierarchical navigation paths
- Help users understand their current location
- Provide quick navigation to parent pages
- Display the site structure
- Aid in SEO and user experience

## Accessibility

- Breadcrumbs have proper semantic structure
- Screen readers can navigate the breadcrumb trail
- Keyboard navigation is supported
- Proper ARIA attributes are applied
- Current page is clearly identified

## Design Guidelines

- Keep breadcrumb labels concise and clear
- Use consistent separators throughout the site
- Ensure sufficient contrast for readability
- Consider mobile responsiveness
- Don't make breadcrumbs too deep (3-5 levels max)
