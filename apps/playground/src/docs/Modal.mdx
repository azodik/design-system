# Modal Component

The Modal component provides overlay dialogs for displaying important information, forms, or interactive content that requires user attention.

## Introduction

The Modal component includes several complementary features to handle various overlay and focused interaction scenarios:

- Modal: the main wrapper for creating overlay dialogs with backdrop and content management.
- Modal Content: structured content areas with header, body, and footer sections for organized layouts.
- Modal State Management: controlled open/close state handling with backdrop click and escape key support.
- Modal Sizing: multiple size options (sm, md, lg, xl) to accommodate different content types and screen sizes.
- Modal Customization: flexible styling, positioning, and animation options for different design needs.
- Modal Accessibility: proper ARIA attributes, focus management, keyboard navigation, and screen reader support.

### When to use
- Display important information or confirmations
- Show forms that need focused attention
- Display detailed content without navigation
- Confirm destructive actions
- Show image galleries or media

## Form Modal

<CodeBlockWithPreview 
  title="Modal with Form"
  preview={
    <div className="p-4 border rounded-lg bg-gray-50">
      <h3 className="text-lg font-semibold mb-2">Form Modal Example</h3>
      <p className="text-sm text-gray-600">This would show a modal with form fields</p>
    </div>
  }
  code={`import { Modal, ModalHeader, ModalFooter, Button, Input } from '@azodik/ui';
import { useState } from 'react';

export const FormModal = () => {
  const [isOpen, setIsOpen] = useState(false);
  const [formData, setFormData] = useState({ name: '', email: '' });

  const handleSubmit = (e) => {
    e.preventDefault();
    setIsOpen(false);
  };

  return (
    <>
      <Button onClick={() => setIsOpen(true)}>
        Add User
      </Button>
      <Modal isOpen={isOpen} onClose={() => setIsOpen(false)}>
        <ModalHeader>
          <h2>Add New User</h2>
        </ModalHeader>
        <div>
          <form onSubmit={handleSubmit}>
            <div>
              <label>Name</label>
              <Input 
                value={formData.name}
                onChange={(e) => setFormData({...formData, name: e.target.value})}
              />
            </div>
            <div>
              <label>Email</label>
              <Input 
                type="email"
                value={formData.email}
                onChange={(e) => setFormData({...formData, email: e.target.value})}
              />
            </div>
            <ModalFooter>
              <Button variant="outline" onClick={() => setIsOpen(false)}>
                Cancel
              </Button>
              <Button type="submit">
                Add User
              </Button>
            </ModalFooter>
          </form>
        </div>
      </Modal>
    </>
  );
};`}
/>
