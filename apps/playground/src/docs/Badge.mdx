# Badge Component

The Badge component displays small status indicators, labels, counts, and notifications. It's perfect for highlighting important information or showing quantities.

## Overview

Badges are small visual indicators that provide additional context or information. They can be used to show status, counts, labels, or notifications.

### When to use
- Display status indicators (success, warning, error)
- Show counts or quantities
- Label content with categories or tags
- Highlight important information
- Show notification counts

## Accessibility

- Badges have appropriate color contrast
- Text content is readable at all sizes
- Screen readers can access badge content
- Color is not the only way to convey information

## Design Guidelines

- Use appropriate variants for different contexts
- Keep badge text concise and clear
- Ensure sufficient contrast for readability
- Use consistent sizing across your application
- Consider the background when choosing colors
