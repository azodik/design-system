# Badge Component

The Badge component displays small status indicators, labels, counts, and notifications. It's perfect for highlighting important information or showing quantities.

## Introduction

The Badge component includes several complementary features to handle various status and notification scenarios:

- Badge: the main wrapper for displaying small status indicators and labels.
- Badge Variants: different visual styles (primary, secondary, success, warning, error, info, neutral) to indicate status.
- Badge Sizing: multiple size options (sm, md, lg) for different use cases and contexts.
- Badge Customization: flexible border radius and styling options for different design needs.
- Badge Icons: optional icon support to enhance visual communication and context.
- Badge Actions: optional remove functionality for dismissible badges and interactive elements.

### When to use
- Display status indicators (success, warning, error)
- Show counts or quantities
- Label content with categories or tags
- Highlight important information
- Show notification counts

## Accessibility

- Badges have appropriate color contrast
- Text content is readable at all sizes
- Screen readers can access badge content
- Color is not the only way to convey information

## Design Guidelines

- Use appropriate variants for different contexts
- Keep badge text concise and clear
- Ensure sufficient contrast for readability
- Use consistent sizing across your application
- Consider the background when choosing colors
