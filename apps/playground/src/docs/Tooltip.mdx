# Tooltip Component

The Tooltip component provides contextual information that appears on hover or focus. It's perfect for providing additional details without cluttering the interface.

## Overview

Tooltips are small popup windows that appear when users hover over or focus on an element. They provide additional context, explanations, or helpful information without taking up permanent space in the UI.

### When to use
- Provide additional context or explanations
- Show helpful hints or tips
- Explain abbreviations or technical terms
- Display additional information about icons or buttons
- Guide users through complex interfaces

## Accessibility

- Tooltips are announced to screen readers
- Keyboard navigation is supported
- Focus management is handled correctly
- Proper ARIA attributes are applied
- Color is not the only way to convey information

## Design Guidelines

- Keep tooltip content concise and helpful
- Use appropriate positioning to avoid clipping
- Ensure sufficient contrast for readability
- Don't rely solely on tooltips for critical information
- Consider mobile touch interactions
